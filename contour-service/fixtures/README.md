# Fixtures

## DEM fixture

A small DEM file is intentionally not committed. Download a small public DEM and place it at:

- `fixtures/dem/sample_dem.tif`

The local `dem_catalog.json` expects that path.

Example source options:
- USGS 1/3 arc-second DEM (GeoTIFF)
- Any tiny COG/GeoTIFF in EPSG:4326 covering the sample AOI in San Francisco
